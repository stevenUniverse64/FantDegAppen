# FantDeg Appen

<img src="Media/FantDegLogo.png" width="400">

## Generell informasjon
Dette er en applikasjon som er utviklet under forløpet av bachelor i IT & Informasjonssystemer. 

### Teknologi
- **Backend:** Node.js, Express
- **Frontend:** React (Vite)
- **Annet:** REST API, CORS, Leaflet

### Funksjonalitet
FantDeg er en fullstack webapplikasjon som henter og presenterer sanntids kameradata fra Statens vegvesen. Applikasjonen består av en backend som henter og filtrerer data, og en frontend som viser kameraene til brukeren på en oversiktlig måte.

## Kom i gang

### Krav
- Node.js

### Installasjon
```bash
# Installer avhengigheter (Kjøres i terminal fra rotmappe)
cd backend && npm install
cd ../frontend && npm install

# Start backend
cd backend && npm start

# Start frontend
cd frontend && npm run dev
```
